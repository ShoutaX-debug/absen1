# **App Name**: GeoFace Attend Pro

## Core Features:

- User Login: Secure user authentication using JWT.
- Selfie Attendance: Capture a selfie for attendance using the mobile app's camera.
- GPS Location Tracking: Automatically capture GPS location for attendance validation.
- Office Radius Validation: Verify if the user is within the defined office radius using the Haversine formula.
- Attendance History: Display a personal attendance history within the mobile app.
- Admin Dashboard: Web interface for admin to manage employees, view attendance data, and generate reports.
- AI Anomaly Detection: Use a tool that leverages historical attendance patterns to flag unusual attendance activities.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and reliability, echoing the serious nature of timekeeping.
- Background color: Very light gray (#F5F5F5) for a clean, neutral backdrop that emphasizes data clarity and reduces visual fatigue.
- Accent color: Vivid orange (#FF9800) for interactive elements and call-to-action buttons, creating a sense of urgency.
- Body and headline font: 'Inter', a sans-serif typeface, for a clean, modern, objective style that conveys a sense of precision, as well as easy readability.
- Use material design icons for a consistent and modern look.
- Clean, minimalist layout with clear visual hierarchy for ease of use.
- Subtle transitions and animations to provide feedback on user interactions.
